# Web App — Ortho.ia

⚠️ **Reconstitué sans captures d'écran.** Aucune image de l'app n'a été fournie ; ce kit est une proposition cohérente avec la voix de marque et les conventions métier (anamnèse / observations / scores / conclusion). À recalibrer dès qu'on a accès aux écrans réels.

## Écrans

- `Login` — connexion simple
- `Dashboard` — liste des bilans (brouillons / à relire / validés)
- `BilanForm` — saisie d'un bilan en sections
- `ReportPreview` — aperçu du compte-rendu généré, prêt à exporter

## Composants partagés

- `AppShell.jsx` — layout sidebar + header
- `Sidebar.jsx`
- `AppHeader.jsx`
- `AppPrimitives.jsx` — Button, Input, Badge, Card, IconButton

## Navigation

L'`index.html` est une démo cliquable : connexion → dashboard → ouverture d'un bilan → génération → aperçu rapport.
