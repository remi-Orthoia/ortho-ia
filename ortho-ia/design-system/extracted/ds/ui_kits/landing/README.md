# Landing — Ortho.ia

Site marketing principal. **Direction A par défaut** ; passer `data-direction="B"` sur `<html>` pour basculer.

## Composants

- `Header.jsx` — barre fine, logo + nav + CTA primaire
- `Hero.jsx` — promesse principale, mockup décor, double CTA
- `LogosBar.jsx` — réassurance "comme vous, des centaines de cabinets"
- `Steps.jsx` — 3 étapes : saisir / générer / relire
- `Beneficts.jsx` — bénéfices chiffrés (h/semaine, etc)
- `Quote.jsx` — témoignage orthophoniste
- `Pricing.jsx` — 2 plans (Solo, Cabinet)
- `FAQ.jsx` — questions fréquentes
- `Footer.jsx` — pied de page éditorial
- `Primitives.jsx` — Button, Badge, Container shared

L'index.html assemble une vraie page marketing en haute fidélité.
